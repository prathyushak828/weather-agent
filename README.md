# Weather Agent

A Java-based AI weather agent built with Anthropic Claude and Open-Meteo API.

## Setup
1. Clone the repo
2. Set your API key: `$env:ANTHROPIC_API_KEY = "your-key"`
3. Run: `mvn compile && mvn exec:java`

## Usage
Ask anything like:
- "What is the weather in Austin?"
- "Will it rain in London tomorrow?"
